CSS原理

优先原则
后解析的内容 会覆盖掉之前解析的内容

1.对于同一个选择器：文件执行的顺序是从上往下执行。
例如：

2.同一类型的选择器：从上往下
例如：

3.不同类型的选择器：选择器本身具有优先级
如：*<div<class<id
规则：
先解析 低优先级的再解析高优先级的
